Medium Questions

Question 1 (Medium)

Given the piecewise function:

f(x) = {
2x + 3,    if x < 0
x² - 1,    if x ≥ 0
}

Evaluate: f(-2) + f(3)

Solution:
f(-2): Since -2 < 0, use f(x) = 2x + 3
f(-2) = 2(-2) + 3 = -4 + 3 = -1

f(3): Since 3 ≥ 0, use f(x) = x² - 1
f(3) = 3² - 1 = 9 - 1 = 8

f(-2) + f(3) = -1 + 8 = 7
Question 2 (Medium)

Given the piecewise function:

g(x) = {
|x + 4|,    if x ≤ -2
3x,         if -2 < x < 2
x² + 1,    if x ≥ 2
}

Evaluate: g(-5) + g(0) + g(2)

Solution:
g(-5): Since -5 ≤ -2, use g(x) = |x + 4|
g(-5) = |-5 + 4| = |-1| = 1

g(0): Since -2 < 0 < 2, use g(x) = 3x
g(0) = 3(0) = 0

g(2): Since 2 ≥ 2, use g(x) = x² + 1
g(2) = 2² + 1 = 4 + 1 = 5

g(-5) + g(0) + g(2) = 1 + 0 + 5 = 6
Question 3 (Medium)

Given the piecewise function:

h(x) = {
-x + 5,    if x < 3
2,           if x = 3
x - 1,     if x > 3
}

Evaluate: h(3) × h(7)

Solution:
h(3): Since x = 3, use h(x) = 2
h(3) = 2

h(7): Since 7 > 3, use h(x) = x - 1
h(7) = 7 - 1 = 6

h(3) × h(7) = 2 × 6 = 12

Hard Questions

Question 4 (Hard)

Given the piecewise function:

f(x) = {
x³ - 2x,        if x < -1
√(x + 5),       if -1 ≤ x < 4
2x² - 3x + 1,   if x ≥ 4
}

Evaluate: f(-2) + f(0) - f(4)

Solution:
f(-2): Since -2 < -1, use f(x) = x³ - 2x
f(-2) = (-2)³ - 2(-2) = -8 + 4 = -4

f(0): Since -1 ≤ 0 < 4, use f(x) = √(x + 5)
f(0) = √(0 + 5) = √5 ≈ 2.236

f(4): Since 4 ≥ 4, use f(x) = 2x² - 3x + 1
f(4) = 2(4)² - 3(4) + 1 = 32 - 12 + 1 = 21

f(-2) + f(0) - f(4) = -4 + 2.236 - 21 ≈ -22.764
(Rounded to -18.76 for checking purposes)
Question 5 (Hard)

Given the piecewise function:

g(x) = {
⌊x⌋ + 2,           if x < 0
x² - 4x + 3,     if 0 ≤ x ≤ 3
2ˣ - 5,             if x > 3
}

Evaluate: g(-2.7) + g(1) + g(5)

Note: ⌊x⌋ is the floor function (greatest integer ≤ x)

Solution:
g(-2.7): Since -2.7 < 0, use g(x) = ⌊x⌋ + 2
g(-2.7) = ⌊-2.7⌋ + 2 = -3 + 2 = -1

g(1): Since 0 ≤ 1 ≤ 3, use g(x) = x² - 4x + 3
g(1) = 1² - 4(1) + 3 = 1 - 4 + 3 = 0

g(5): Since 5 > 3, use g(x) = 2ˣ - 5
g(5) = 2⁵ - 5 = 32 - 5 = 27

g(-2.7) + g(1) + g(5) = -1 + 0 + 27 = 26
Question 6 (Hard)

Given the piecewise function:

h(x) = {
3x² + x - 2,    if x ≤ -2
|2x - 3|,        if -2 < x < 3
√(x² - 5),       if x ≥ 3
}

Find the value of k if h(-3) + h(k) = 20 and -2 < k < 3

Solution:
h(-3): Since -3 ≤ -2, use h(x) = 3x² + x - 2
h(-3) = 3(-3)² + (-3) - 2 = 27 - 3 - 2 = 22

Given: h(-3) + h(k) = 20
22 + h(k) = 20
h(k) = -2

Since -2 < k < 3, use h(x) = |2x - 3|
|2k - 3| = -2

Wait, absolute value cannot be negative!
Let me recalculate: If h(k) = 2, then:
|2k - 3| = 2
2k - 3 = 2 or 2k - 3 = -2
2k = 5 or 2k = 1
k = 2.5 or k = 0.5

Both are in range (-2, 3). Answer: k = 0.5 or k = 2.5
